1. Clone this entire repository and connect that GIT to Cloudflare Pages.

2. Build command: "npm run"
// This ensures that packages are installed from the package.json file in the root directory.

3. Set the compatibility flags for production to: "nodejs_compat"

4. Bind the following Cloudflare KV namespace bindings:

KV namespace		Variable name
coolfrog.users ------>	COOLFROG_USERS
